# autodealimg
在线处理图片，缩略，背景色，水印
在线处理图片说明  

请求格式： 图片url?长X宽m[模式数值]bc[颜色值]w[水印位置值]-[水印透明度]-水印图片网络url
实例 : http://xxxx.jpg?900x900m3bc200,120,50w5-7-https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png

模式说明:
m1 : 固定大小缩略,图可能会有所变形
m2 : 等比例缩放
m3 : 缩放填充
m4 : 从左上角开始裁减指定宽高

水印位置值 ： 
1 ：左上角水印
2 ：上居中水印
3 ：右上角水印
4 ：左居中水印
5 ：居中水印
6 ：右居中水印
7 ：左下角水印
8 ：下居中水印
9 ：右下角水印

水印透明度：
1-9 数值越大越明显

bc200,120,50
填充图片的背景颜色
